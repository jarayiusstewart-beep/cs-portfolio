## Text-Based RPG Game (Python)

### Description
A text-based RPG where the player makes choices through typed commands to complete a short adventure. The goal is to rescue a princess from the goblin king, with different outcomes based on player decisions.

### Tools I Used
- Python
- Command-line input/output
- Functions
- Loops and conditional statements

### What I Learned
- How to organize a program using functions
- How to control game flow with loops
- How to handle user input and choices
- How to create multiple endings based on decisions
